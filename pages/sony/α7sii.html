Sony α7S II

詳細規格
推出日期 約 2015 年 9 月
感光元件像素 1220 萬像素
感光元件大小 35.6 x 23.8mm
感光元件種類 CMOS
鏡頭接環 Sony E 接環，可透過 LA-EA3 / LA-EA4 接環使用 Sony Alpha 鏡頭
自動對焦情況 Fast Intelligent 對比感測系統多點對焦 ( 169 點 ) / 中央 / 靈活對焦點 ( 細 / 中 / 大 ) / 局部眼部對焦功能
最大解析度 4240 x 2832
影片解析度 4K : 3840 x 2160 @ 30/25/24 fps HD : 1920 x 1080 @ 60/50/25/24 fps ; 1280 x 720 @ 30/25p HFR : 1920 x 1080 @ 120 fps
影片種類 XAVC S / AVCHD / MP4
曝光模式
  • 1. 支援光圈先決
  • 2. 支援快門先決
  • 3. 支援手動曝光
    • 場景模式 肖像、風景、微距拍攝、體育活動、日落、夜景肖像、夜景、夜景手持拍攝、防止影像模糊
    測光模式 1200 區域測光支援 -3EV 至 20EV多區、中央、定點
    曝光補償 -5EV 至 +5EV 以 1/3EV 級調整
    白平衡設定 自動、日光、陰影、陰天、白熾燈、螢光燈 ( 4 種 )、閃光燈、自訂、水底、自訂 Kelvin 值支援手動白平衡設定
    連拍 2.5fps / 5fps ( 速度先決連拍模式 )
    快門速度 1/8000 至 30 秒，支援 B 快門
    ISO 感光值 100 - 102400，可擴展至 ISO 50 或 ISO 409600（拍片時最低 ISO 200）
    LCD螢幕 3 吋 (1228800 像素) TFT LCD 熒光幕 ，視野率為 100 %
    儲存介面 SD / SDHC / SDXC / Memory Stick PRO Duo / Memory Stick PRO-HG Duo / Memory Stick XC-HG Duo
    電池種類 NP-FW50
    電池壽命 310 張
    機身重量 584 克
    重量(含電池) 627 克
    體積 126.9 x 95.7 x 60.3mm
    其他 不設 Low Pass Filter 採用 BIONZ X 影像處理器 設有一掃全景模式 HDR 功能 1-5 級 D-Range Optimizer 優化技術 Peaking 手動對焦 S-Log3 Gamma 對焦系統支援最低 -4EV
